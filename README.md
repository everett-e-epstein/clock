# Clock

An analog clock with rotating concentric circles and dynamic time text display.

## Features

- Rotating concentric circles instead of traditional clock hands
- Real-time display showing hour and minute in text form
- Diatype EDU font
- Modern, minimalist design

## How to Use

Simply open `index.html` in a web browser to view the clock.

## Files

- `index.html` - Main HTML file
- `style.css` - External stylesheet
- `fonts/` - Diatype EDU font files
